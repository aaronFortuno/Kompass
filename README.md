# Kompass

Web d'autoaprenentatge d'alemany A1 per a catalanoparlants. SPA estàtica, sense backend, amb progrés transportable.

**Estat:** en construcció. Vegeu [`ARCHITECTURE.md`](./ARCHITECTURE.md) i [`DATA-MODEL.md`](./DATA-MODEL.md) per a l'especificació tècnica.

## Desenvolupament

```bash
npm install
npm run dev
```
